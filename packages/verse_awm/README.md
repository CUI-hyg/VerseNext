# VerseAWM

Autonomous World Model package (JEPA, RSSM, H-JEPA).

Part of the [Verse](../../README.md) framework. Provides non-generative
Joint-Embedding Predictive Architecture (I-JEPA / V-JEPA / H-JEPA) and
generative Recurrent State-Space Model (RSSM, Dreamer-style) implementations
on top of VerseTorch, with EMA target encoders and energy-based losses.
